# Socket-Programming
- java 소켓을 이용한 야구게임 구현
- 프로그램 설명서 참조
## 실행화면
- 서버는 입력받은 숫자에 맞게 nS mB (n=스트라이크 수, m=볼 수)와 남은 횟수 출력
